# Security Policy

## Supported Versions

This repository currently supports security fixes on the latest code in `main`.

| Version | Supported |
| --- | --- |
| `main` | :white_check_mark: |
| other branches | :x: |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use GitHub's private vulnerability report feature:
- [Report a vulnerability](https://github.com/yoichi-aoyama/kirokuflow/security/advisories/new)

Include the following details:
- affected endpoint/file and reproduction steps
- expected vs actual behavior
- impact assessment and any proof of concept

## Response Targets

- Initial response: within 72 hours
- Triage/update: within 7 days
- Fix timeline: based on severity and exploitability
